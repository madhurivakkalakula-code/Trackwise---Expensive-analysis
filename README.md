# TrackWise - Expense Analytics Platform

## Project Description
TrackWise is a web-based expense tracking system that helps users manage daily expenses and analyze spending patterns.

## Technologies Used
- PHP
- MySQL
- HTML, CSS, JavaScript
- Chart.js
- XAMPP

## Features
- Add Expenses
- View Expense List
- Delete Expenses
- Category-wise Analytics
- Monthly Expense Tracking

## How to Run
1. Install XAMPP
2. Place project in htdocs
3. Start Apache & MySQL
4. Open http://localhost/trackwise
